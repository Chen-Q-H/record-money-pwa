# 花销记录功能规格

## 功能概述

### 功能名称
花销记录查看与管理

### 用户故事
作为用户，我希望能够查看我的所有消费记录，按日期分组显示，以便了解我的消费习惯和模式。

### 业务价值
- 提供完整的消费历史视图
- 帮助用户分析消费行为
- 支持财务决策和预算规划

## 技术规格

### 数据结构
```typescript
interface ExpenseRecord {
  id: string;
  amount: number;
  category: ExpenseCategory;
  date: string;
  description?: string;
}

type ExpenseCategory = '餐饮' | '购物' | '交通' | '娱乐';
```

### 页面组件
```typescript
// ExpensesPage.tsx
interface ExpensesPageProps {
  userData: UserData;
  onNavigate: (page: AppPage) => void;
  currentPage: AppPage;
}
```

### 功能特性
1. **按日期分组**：消费记录按日期自动分组显示
2. **分类图标**：每个消费类别显示对应的emoji图标
3. **统计信息**：显示总消费笔数和每日消费合计
4. **空状态处理**：无记录时显示友好的提示信息
