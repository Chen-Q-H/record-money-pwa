# SVG作物图案设计规范

## 功能概述

### 功能名称
可爱作物SVG图案和生长动画系统

### 用户故事
作为用户，我希望看到可爱的SVG作物图案，而不是简单的emoji，并且能够观察到作物从幼苗到成熟的生长动画过程。

### 业务价值
- 提供更美观、专业的视觉体验
- 增强游戏的趣味性和沉浸感
- 通过动画效果增强用户成就感

## 技术规格

### SVG图案设计要求

#### 胡萝卜图案设计
**三个阶段设计：**
1. **幼苗阶段**：小绿苗，简单线条
2. **成长阶段**：橙色胡萝卜身体，绿色叶子
3. **成熟阶段**：完整的胡萝卜，叶子更茂盛

**颜色方案：**
- 胡萝卜身体：橙色 (#FFA500)
- 叶子：绿色 (#228B22)
- 背景：透明

#### 番茄图案设计
**三个阶段设计：**
1. **幼苗阶段**：小绿苗
2. **成长阶段**：红色圆形番茄，绿色小叶子
3. **成熟阶段**：饱满的番茄，叶子更完整

**颜色方案：**
- 番茄：红色 (#FF0000)
- 叶子：绿色 (#228B22)
- 茎：棕色 (#8B4513)

#### 玉米图案设计
**三个阶段设计：**
1. **幼苗阶段**：细长绿苗
2. **成长阶段**：黄色玉米棒雏形
3. **成熟阶段**：完整的玉米棒，绿色外皮

**颜色方案：**
- 玉米粒：黄色 (#FFD700)
- 外皮：绿色 (#228B22)
- 穗：棕色 (#8B4513)

### 动画设计要求

#### 生长动画效果
- **平滑过渡**：使用CSS transition实现阶段间的平滑过渡
- **缩放效果**：幼苗到成熟有适当的缩放变化
- **颜色渐变**：颜色从浅到深渐变
- **旋转效果**：轻微旋转增强生长感

#### 动画参数
```css
.crop-svg {
  transition: all 0.5s ease-in-out;
  transform-origin: center;
}

.crop-svg:hover {
  transform: scale(1.1) rotate(5deg);
}
```

## 验收标准

### 视觉验收
- ✅ 三种作物都有三个阶段的SVG图案
- ✅ SVG图案可爱、简洁、符合小猪主题
- ✅ 颜色搭配协调，符合实际作物颜色
- ✅ 图案在不同尺寸下清晰可见

### 动画验收
- ✅ 生长动画流畅自然
- ✅ 阶段过渡平滑无闪烁
- ✅ 悬停效果增强交互体验
- ✅ 动画性能良好，不卡顿

### 技术验收
- ✅ SVG代码优化，文件大小合理
- ✅ 响应式设计，适配不同屏幕
- ✅ 与现有作物系统无缝集成
- ✅ 类型安全，TypeScript支持完整
